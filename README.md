# Agent OS Knowledge Vault

This vault stores agent context, research, drafts, and shared knowledge.

## Structure

- **agents/** - Agent-specific work folders
  - alex-research/ - Research findings and sources
  - maya-content/ - Blog drafts and content
  - jordan-strategy/ - Marketing plans and campaigns
  - dev-projects/ - Code notes and project docs
  - sam-social/ - Social posts and performance tracking
- **shared/** - Cross-agent knowledge base
  - Brand voice, target audience, product info
- **memory/** - System-wide learnings and decisions

## Usage

Agents read/write via obsidian-cli or direct file operations.
